3.1 Blocks

Description

Colette comes with basic styles for block elements. Gives a white background, a full width, a relative position and align the text to the left. Can provides optionnal title and footer. The SVG in the title is optionnal too. You can add a 'box' class to Block, it will give it a border and paddings. It's optionnal too. It can be modified by -stretch, it'll be useful to make it stretch in height.
